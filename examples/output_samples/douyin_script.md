# 🎵 抖音视频脚本样例

**主题**: MCP协议让AI更强大
**模式**: rewrite
**总时长**: ~45秒

---

[镜头1] 近景 - 博主表情惊讶
[画面] 博主看着屏幕，突然抬头看向镜头，瞪大眼睛
[字幕] "绝了！这个协议让AI开发效率提升10倍"
[时长] 0-3秒
[BGM] 节奏感强的电子音乐，突然停顿制造悬念

[画面] 左侧：博主疯狂敲键盘，表情痛苦（代表传统开发）

[时长] 3-8秒
[转场] 快速切换，带"咻"的音效

[镜头3] 特写 - 屏幕录制
[画面] 展示MCP配置代码，只有短短几行
[字幕] "就这几行代码，AI就能操作文件了"
[时长] 8-15秒
[特效] 代码高亮，关键行用红框标注

[镜头4] 中景 - 手势展示
[画面] 博主用手势展示积木拼接的动作
[字幕] "标准化接口，像搭积木一样简单"
[时长] 15-22秒

[镜头5] 快切镜头 - 平台Logo快闪
[画面] Claude、Cursor、其他AI工具的Logo快速闪过
[字幕] "一次开发，到处运行"
[时长] 22-28秒
[转场] 快闪效果，配合"滴滴滴"音效

[镜头6] 近景 - 博主认真脸
[画面] 博主直视镜头，表情认真
[字幕] "这就是MCP协议，AI时代的HTTP"
[时长] 28-35秒

[镜头7] 近景 - 博主微笑
[画面] 博主面对镜头，竖起大拇指
[字幕] "关注我，分享更多AI开发神器！"
[时长] 35-40秒

[镜头8] 定格画面
[画面] 博主手指上方"关注"按钮位置
[字幕] "点关注，不迷路 👆"
[时长] 40-45秒

[BGM] 音乐渐强，结束
[整体风格] 快节奏、信息密度高、口语化
